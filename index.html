This link is load a custom modal window with 700px width, 2000 ms animation speed, the title is not visible and modal class: blue

Call like this in js(js/global.js line 2): $('.modal').modal();
.modal is the class name, what you gave it to the link
HTML code: <a href="test.html" title="This is the title" class="modal">This is a simple modal link</a>
The href attribute can be a url(e.g.: test.html) or a div id(e.g.: #loadthis)
The title attribute value show in the modal header

Avaible options:
$('.modal').modal({width:400,showSpeed:500,closeSpeed:500,title:true,skin:"default"});

  • width: the modal window width, in px
  • height: the modal window max-height, in px
  • showSpeed: the number of milliseconds to run the animation
  • closeSpeed: the number of milliseconds to run the animation
  • autoOpen: true or false.
  • autoClose: the number of milliseconds to close automatically
  • position: e.g. {top:"200px",right:"10px"}.
  • overlay: true or false.
  • title: true or false.
  • draggable: true or false.
  • skin: skin class name.

If you want a skin, just add a new class to style.css, like this:
.modal_window.red {
background:red;
border:2px solid blue;
}
